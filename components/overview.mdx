## Install

Install React Chron and the Chron.js loader from [npm public registry](https://www.npmjs.com/package/@chron/react-chron) using your favourite package manager.

<CodeGroup>
```bash npm
npm install @chron/react
```

```bash pnpm
pnpm install @chron/react
```
</CodeGroup>

## Environment Variables

Add your publishable key to your environment variables.

<CodeGroup>
```bash .env.local
CHRON_API_KEY=YOUR_CHRON_API_KEY
```
</CodeGroup>

## Configure

Configuration should happen as early as possible in your application's lifecycle.

<CodeGroup>
```tsx index.tsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.tsx'
import './index.css'
import { ChronProvider } from '@chron/react'

// Import your publishable key
const CHRON_API_KEY = import.meta.env.CHRON_API_KEY

if (!CHRON_API_KEY) {
  throw new Error("Missing Publishable Key")
}

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <ChronProvider publishableKey={CHRON_API_KEY}>
      <App />
    </ChronProvider>
  </React.StrictMode>,
)
```
</CodeGroup>
